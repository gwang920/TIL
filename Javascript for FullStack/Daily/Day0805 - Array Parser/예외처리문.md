# 예외처리문

## 1. throw 문

```
예외를 사용할 떄 throw문을 사용한다.
throw문은 사용자가 정의한 예외를 던질 수 있다.
```

```javascript
사용자정의로 throw문을 정의한 코드
----------------------------------------------------
function sum(x,y){
	if(typeof x !=='number' ||
	typeof y!=='number'
	){
		throw '숫자가 아닙니다'
	}
	return x+y;
}

console.log(sum('a,',2));
----------------------------------------------------------------------------------------
[결과]
uncaught throw
```



## 2. try catch문

```
앞서 봤던 throw는 에러를 잡아줄 수 있는 구문이 필요하다.
그게 바로 try catch문이다.

예제로 살펴보자
```

```javascript
함수 f2() throw문이 없었다면
아래와 같은 결과가 나타난다. 

will : f1
f1 start
f2 start
f2 end
f1 end
did : f1

하지만 throw문이 포함되어 있기에
코드의 실행결과는 f2 start에서 정지한다.
에러를 잡지못해 f2가 먼저 파괴되고 
순차적으로 f2를 호출한 f1, f1을 호출한 시점 이후는 전부파괴되는 것이다.
----------------------------------------------------------------------------------------
fucntion f2(){
	console.log('f2 start');
    throw '에러';
	console.log('f2 end');
}

function f1(){
	console.log('f1 start');
	f2();
	console.log('f1 end');
}

console.log('will : f1');
f1();
console.log('did : f1');
----------------------------------------------------------------------------------------
[결과]
will : f1
f1 start
f2 start
```

- 위 예제를 try catch로 잡아보자

```javascript

정상적으로 모든 코드가 실행됨을 확인할 수 있다.
----------------------------------------------------------------------------------------
function f2(){
	console.log('f2 start');
    throw '에러';
	console.log('f2 end');
}

function f1(){
	console.log('f1 start');
	try{
		f2();
	}catch(e){
		console.log(e);
	}
	console.log('f1 end');
}

console.log('will : f1');
f1();
console.log('did : f1');
----------------------------------------------------------------------------------------
[결과]
will : f1
f1 start
f2 start
에러
f1 end
did : f1
```

- try catch 문에 따라 실행결과가 어떻게 변하는지 확인해 보자

```javascript
f2()에서 에러를 잡아주지 않아 f1 end는 출력되지 않고 파괴된다.
연쇄적으로 따라가 f1()을 호출한 놈으로 되돌아가고
이때 try catch문이 존재하므로 에러를 잡아줘 did : f1은 출력됨을 확인할 수 있다.
----------------------------------------------------------------------------------------
function f2(){
	console.log('f2 start');
    throw '에러';
	console.log('f2 end');
}

function f1(){
	console.log('f1 start');
	f2();
	console.log('f1 end');
}

console.log('will : f1');
try{
 f1();   
}catch(e){
    console.log(e);
}
console.log('did : f1');
----------------------------------------------------------------------------------------
[결과]
will : f1
f1 start
f2 start
에러
did : f1
```
