#### I. Ajax

- Ajax는 서버와 데이터를 교환하는 기술

- 비동기적으로 HTML 페이지를 업데이트

- 즉, 전체 페이지를 다시 적재하지 않고, 웹 페이지의 일부를 업데이트 할 수 있다. 

  ##### 	  ex) 검색창 자동완성 기능, 실시간 검색 순위
