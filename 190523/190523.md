## 2) 클래스변수와 인스턴스변수

* 클래스변수와 인스턴스변수의 차이를 느껴보자
  * 카드게임
    * 속성 (무늬, 숫자) : 인스턴스변수 -> 7,4,3 / 하트,클로버 등 변화가능한 요소
    * 속성 (폭, 높이) : 클래스변수 -> 카드에서 유지 되어야할 요소

``` java
class Card{
	String kind;          // 무늬
	int number;			  // 숫자
	
	static int width = 100;  // 폭
	static int height = 250; // 높이
}
```
