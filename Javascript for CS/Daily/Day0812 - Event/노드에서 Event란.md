# 노드에서 Event란?

```
노드에서 대부분의 이벤트는 비동기 방식을 처리하고, 비동기 방식으로 이벤트를 전달한다.
이벤트는 한쪽에서 다른 쪽으로 알림메시지를 보내는 것과 비슷하다.

이런 이벤트를 받을 수 있도록 노드에서는 EventEmitter라는 것이 만들어져있다. 

노드의 객체는 EventEmitter를 상속받을 수 있고, 상속 받은 후에는 EventEmitter 객체의 on()과 emit() 메소드를 사용할 수 있다.

on() 메소드는 이벤트가 전달될 객체에 이벤트 리스너를 설정하는 역할을 하는데 이 리스너 함수는 객체로 전달된 이벤트를 받아 처리할 수 있다.

어떤 객체든 상속을 통해 EventEmitter가 될 수 있다.
```
