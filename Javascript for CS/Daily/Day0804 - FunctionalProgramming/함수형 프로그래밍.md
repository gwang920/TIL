# 함수형 프로그래밍

## 1. 함수형 프로그래밍

```
1. 인풋과 아웃풋이 있다.
결과물을 반환(동일한 인풋 동일한 아웃풋)
2. 외부 환경과 독립적이다.
다른 것들을 참조하지 않는다.
(외부 변수에 영향을 받지 않는다.)

시간차에 따른 오류

문제의 소지가 있는 일을 하지 않는 코딩 방식
부작용이 없다.
```
