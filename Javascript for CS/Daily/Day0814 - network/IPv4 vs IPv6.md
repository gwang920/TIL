# IPv4 vs IPv6

```
지금 우리가 쓰고있는 ip는 IPv4 주소체계인데 43억개로 주소 개수를 할당하기엔 턱없이 부족하다. 이를 보완하기 위해 IPv6가 생겨났다.

최근에 IOT나 자율주행, 스마트폰 등 초연결시대가 되면서 다양한 기기와 데이터를 주고 받아야하기에 IP를 할당해야할 일이 많아졌다.
새로 등장한 이 IPv6가 이에 활용되고 개수가 거의 무한하기에 IPv4의 문제를 확실히 보완한다.
```



```
IPv4  - 32bit 체계 8bit씩 4개의 옥텟으로 구성된다.
      - 네트워크 ID(해당 컴퓨터가 소속 된 네트워크에 배정된 이름)와
 	    HOST ID(해당 컴퓨터 한대에 배정된 이름)로 구분한다.
      - 8비트씩 4부분으로 10진수로 표시
        ex) 202.30.64.22
        
IPv6 - 128bit 주소 체계로 이루어져있어 Ipv4보다 4배나 많은 정보를 수용할 수 있다.
     - 16비트씩 8부분으로 16진수로 표시
       ex) 2001:avcd:ffff:0000:0000:ffff:1111
```
