# II. CSS 기능

#### :nth-child(), :nth-last-child()

- 특정 순서에 있는 요소를 선택할 때 사용
- :nth-child()  앞에서 부터  :nth-last-child()  뒤에서 부터

```html
x:nth-child(an+b)
- a,b는 정수 / 0과 음수도 가능

ex) table 1열을 가운데 정렬, 녹색 배경

td:nth-child(4n+1){
		text-align:center;
		background:green;
	}

```
